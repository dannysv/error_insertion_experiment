este trabalho lem o intuito de analisar e viabilidade econômica da implantação de uma nano cervejaria associada a un restaurante cm niterói rj paro isso fol feito um levantamento dó consumo médio mensal de cervejas especiais no município então foi estipulado em valor base para os cálculos a partir deste valor base foi feita uma pesquisa para encontrar um valor de implementação da cozinha para esta produção também foram feitos os cálculos dos custos variáveis do processo cómo custos cem insumos e custos energéticos após estas cálculos fei feita uma avaliação utilizando matematica financeira para determinar a viabilidade econômica desta implantação
